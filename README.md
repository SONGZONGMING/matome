- SVC： https://github.com/SONGZONGMING/SVC

深層学習ベースの歌声変換ツールである so-vits-svc を使用し、独自の音声モデルを制作しました。

制作にあたっては、音声素材の整理、ノイズ除去、データセット作成、モデル学習、推理、音声調整まで一連の工程をやりました。

完成したモデルを用いて楽曲の歌声変換を行い、AI音声技術を活用した音楽作品を制作しました。
